Les mots cassés

Qu'on raccommode,

Les mots lassés

Sur la commode ;

 \
Les mots clairs, bleus,

Soudain frémissent,

Les mots sableux

Sous la dent crissent.

 \
Droits, foudroyés,

Les mots d'orage

Et dévoyés

Les mots garage.

 \
Les mots entiers

Comme vandales

Sur les sentiers

Juste en sandales.

 \
Les mots pour toi,

Les mots sur terre,

Trouver son toit,

Dire ou se taire.
