Les mots cadeau

Tombés d'un livre,

Les mots fardeau

Pour dire vivre.

 \
Les mots voyous,

Les mots insultes,

Les mots youyous,

Et les mots cultes.

 \
Les mots mangés,

Les mots acides,

Les mots rangés,

Les mots placides.

 \
Un mot pour toi

Qui te retrouve,

Les mots sans toit

Quand rôde louve.

 \
Les mots costauds,

Les mots fragiles,

Les mots étaux,

Les mots argiles.

 \
Les mots pardons,

Les mots qui pèsent,

Les mots gardons,

Les mots s'apaisent !
